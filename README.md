# Student-Hub

Full-stack student dashboard built with React, Vite, Node.js, Express, and MongoDB.

## Features
- Student dashboard
- Backend API
- Frontend UI
- Full-stack project structure

## Tech Stack
- Frontend: React, Vite, JavaScript, CSS
- Backend: Node.js, Express
- Database: MongoDB

## Project Structure
- frontend
- backend

## Run Locally

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
node server.js
